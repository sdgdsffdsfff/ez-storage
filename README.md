EZ Storage
===
### 数据存储模块

 =======================================================

##功能

1. 实现JDBC的各类CURD操作

##使用

        <groupId>com.ecfront</groupId>
        <artifactId>ez-storage</artifactId>
        <version>0.2.7</version>
    </dependency>
